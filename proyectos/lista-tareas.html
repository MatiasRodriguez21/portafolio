Descripción

Este proyecto es una aplicación de lista de tareas que permite a los usuarios agregar, eliminar y gestionar sus tareas diarias de manera eficiente.

Funcionalidades principales

  • Agregar Tareas: incorporar nuevas tareas a la lista.
  • Eliminar Tareas: quitar tareas completadas o no deseadas.
  • Modo Nocturno: interfaz adecuada para entornos oscuros.
  • Interfaz Intuitiva: diseño simple y fácil de usar.

Tecnologías

Node.js Express HTML CSS JavaScript

Requisitos

  • Node.js
  • Express

Mi aporte

  • Diseño de UI simple y accesible con modo nocturno.
  • Rutas básicas en Express para gestión de tareas (si se usa backend).
  • Organización del código y buenas prácticas de Git.

Aprendizajes

  • Enrutamiento y middleware en Express.
  • Patrones simples de CRUD para tareas.
  • Optimización de UX con foco en claridad y velocidad.
