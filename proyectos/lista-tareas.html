Sistema de Turnos Online

Aplicación web para gestionar reservas en instituciones.

Ver sitio Ver en GitHub
  • Rol: Full Stack Jr
  • Año: 2025
  • Equipo: Individual
  • Estado: En desarrollo

Descripción

Plataforma para gestionar turnos de manera online con creación de usuarios, calendarios, recordatorios y panel de administración. Permite definir disponibilidad por profesional/servicio y evitar solapamientos.

Funcionalidades principales

  • Registro e inicio de sesión con validaciones.
  • Gestión de agenda por profesional y por servicio.
  • Reservas, cancelaciones y reprogramaciones.
  • Dashboard con métricas y búsqueda de turnos.
  • Notificaciones (email) para confirmación y recordatorio.

Tecnologías

FastAPI PostgreSQL React TypeScript Docker (dev)

Mi aporte

  • Diseño de esquema de base de datos y endpoints REST.
  • UI (formularios, calendario, tablas) en React.
  • Autenticación y autorización por roles.
  • Deploy de ambiente demo (Docker Compose).

Aprendizajes

  • Reglas de solapamiento de turnos.
  • Manejo de formularios complejos en React.
  • Buenas prácticas en FastAPI.
